# Обработка изображений

Приложение для обработки изображений с графическим интерфейсом, разработанное в рамках ознакомительной практики лета 2026 года в ИКИТ СФУ, кафедра программной инженерии.

## Описание

Приложение предоставляет базовый набор функций для обработки изображений, включая работу с цветовыми каналами, применение фильтров, изменение яркости и рисование геометрических фигур.

## Возможности

### Работа с файлами
- **Загрузить** - открытие изображения из файловой системы
- **Сохранить** - сохранение изменений в текущий файл
- **Сохранить как** - сохранение изображения в новый файл с выбором формата
- **Сделать снимок с веб-камеры** - захват изображения с камеры в реальном времени

### Работа с цветовыми каналами
- Отображение только **красного** канала
- Отображение только **зелёного** канала  
- Отображение только **синего** канала
- **Сброс** к оригинальному изображению

### Функции обработки
- **Негатив** - инвертирование цветов изображения
- **Яркость** - регулировка яркости от -255 до +255
- **Круг** - рисование закрашенного круга с заданными параметрами
- **Сбросить изменения** - возврат к оригинальному изображению

## Использование

### Загрузка изображения
Для загрузки изображения нажмите кнопку **"Загрузить"** в левой панели приложения. В открывшемся диалоговом окне выберите файл изображения. После загрузки изображение отобразится в левой панели "Оригинал", а его копия для редактирования - в правой панели "Результат".

### Сохранение изображения
- **Сохранить** - сохраняет текущее обработанное изображение в исходный файл
- **Сохранить как** - позволяет сохранить обработанное изображение в новый файл с выбором формата

### Работа с веб-камерой
Нажмите "Сделать снимок с веб-камеры" для захвата изображения. Откроется окно просмотра с камеры. Нажмите **SPACE** для захвата кадра или **ESC** для выхода.

### Работа с цветовыми каналами
- **Красный** - отображает только красный цветовой канал
- **Зелёный** - отображает только зелёный цветовой канал
- **Синий** - отображает только синий цветовой канал
- **Сбросить** - возвращает отображение оригинального изображения

### Функции обработки
- **Негатив** - инвертирует все цвета изображения.
- **Яркость** - открывает диалоговое окно для ввода значения яркости от -255 до +255.
- **Круг** - открывает диалоговое окно для ввода координат центра и радиуса круга. Рисует закрашенный красный круг на изображении.
- **Сбросить изменения** - возвращает изображение в панели "Результат" к состоянию оригинального изображения, отменяя все применённые изменения

## Установка и запуск

### Требования
- Python 3.6+ с поддержкой tkinder
- Установленные зависимости

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск
```bash
python main.py
```
